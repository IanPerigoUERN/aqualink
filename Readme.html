# 🚀 Configuração e Uso do Projeto ⚠️ **Atenção:** Ao importar o projeto, garanta que está utilizando a versão **1.5.1** do SDK para evitar incompatibilidades. ## 📡 Execução com Wi-Fi Antes de compilar o projeto, altere as credenciais da sua conexão Wi-Fi **2.4GHz** ⚠️ (a placa **não suporta 5GHz**). As credenciais estão no arquivo `credentials.h`, localizado na pasta `Conection`. Edite as seguintes linhas: ```cpp #define WIFI_SSID "Seu_SSID" #define WIFI_PASS "Sua_Senha" ``` 🔗 Link para verificar o envio de dados no ThingSpeak: [https://thingspeak.mathworks.com/channels/2840074](https://thingspeak.mathworks.com/channels/2840074) 🎞️ Link com o vídeo de apresentação do projeto: [https://www.youtube.com/shorts/zQpASUA7lro?feature=share](https://www.youtube.com/shorts/zQpASUA7lro?feature=share) --- ## 🛠️ Execução sem Wi-Fi Se **não desejar conectar ao Wi-Fi**, basta compilar o projeto e clicar em **RUN ▶️**, ou então transferir o arquivo `aqualink.uf2` para o armazenamento da sua placa **BitDogLab**. --- ## 🔋 Simulação do Funcionamento 1️⃣ **Após iniciar a placa**, o **logotipo do projeto** será exibido no display. 2️⃣ **Simulação da Energia Solar** ☀️ - **Aumente a luz** na **matriz de LED** para simular o aumento da energia solar. - O LED RGB muda de 🔴 (baterias sem carga) para 🟢 (baterias carregadas). 3️⃣ **Acionamento da Bomba d'Água** 💦 - Mova o joystick no eixo **Y para baixo ⬇️** lentamente. - O **buzzer será acionado 🔊**, simulando o funcionamento da **bomba d'água**. 4️⃣ **Simulação do Nível do Poço** 🏞️ - Mova o joystick no eixo **X para a esquerda ⬅️** para simular a **redução do nível do poço**. - O **buzzer será desligado 📴**, indicando que a **bomba foi desligada**. 5️⃣ **Simulação do Nível da Caixa d'Água** 🚰 - Mova o joystick no eixo **Y para cima ⬆️** para simular a **caixa cheia**. - O **buzzer também será desligado 📴**. 6️⃣ **Baixa Energia Solar** 🌥️ - **Reduza a intensidade da luz** na **matriz de LED**. - O **buzzer será desligado 📴** e o LED RGB voltará para 🔴 vermelho, indicando **baixa carga**. 💡 **Agora é só testar e explorar o funcionamento!** ⚡🚀 --- ## 📄 Sobre Projeto desenvolvido para encerramento da primeira fase do **EmbarcaTech**, utilizando a placa **BitDogLab**.
